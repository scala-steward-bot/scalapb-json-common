## Dependency

Include in your `build.sbt` file

### core

```scala
libraryDependencies += "io.github.scalapb-json" %% "scalapb-json-common" % "0.5.0-M3"
```

for scala-js, scala-native

```scala
libraryDependencies += "io.github.scalapb-json" %%% "scalapb-json-common" % "0.5.0-M3"
```

### macros

```scala
libraryDependencies += "io.github.scalapb-json" %% "scalapb-json-macros" % "0.5.0-M3"
```

```scala
libraryDependencies += "io.github.scalapb-json" %% "scalapb-json-macros-java" % "0.5.0-M3"
```
