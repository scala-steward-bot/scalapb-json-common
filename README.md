## Dependency

Include in your `build.sbt` file

```scala
libraryDependencies += "io.github.scalapb-json" %% "scalapb-json-common" % "0.3.1"
```

for scala-js, scala-native

```scala
libraryDependencies += "io.github.scalapb-json" %%% "scalapb-json-common" % "0.3.1"
```
